# Cerid AI

**Autonomous AI infrastructure for knowledge, operations, and markets.**

Cerid AI builds privacy-first, self-hosted AI systems that run on your own infrastructure — giving you full control over your data, agents, and workflows.

---

## Products

### [Cerid AI Knowledge Companion](https://github.com/Cerid-AI/cerid-ai)
A self-hosted personal AI workspace that unifies multi-domain knowledge bases into a context-aware LLM interface. RAG-powered retrieval across code, finance, projects, and personal artifacts — with 9 intelligent agents, 26 MCP tools, and a full React dashboard. Local-first, privacy-first.

- Multi-provider LLM gateway (Claude, GPT, Grok, Gemini, DeepSeek, Llama)
- Hybrid BM25 + vector search with Neo4j knowledge graph
- File ingestion pipeline supporting 30+ formats
- Hallucination detection, memory extraction, conversation intelligence
- Electron desktop app + Docker deployment

---

## Philosophy

> **Local by default. Private by design.**

We believe AI tools should work for you, not harvest from you. Every Cerid system is built to run entirely on your own hardware, with data that never leaves your control.
